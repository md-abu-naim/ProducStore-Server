## How to started this project

1. First, install the project server:
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```
2. Second, Setup your MongoDB user and password.
3. Third, run the development server:

```bash
nodemon start
# or
nodemon
```
Open [http://localhost:5000](http://localhost:5000) with your browser to see the result.
